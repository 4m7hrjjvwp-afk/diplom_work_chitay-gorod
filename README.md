# diplom_work_chitay-gorod
Автоматизации финального проекта по ручному тестирования  "Читай-город"

Шаги:
Склонировать проект 'git clone https://github.com/4m7hrjjvwp-afk/diplom_work.git'
Установить зависимости
Запустить тесты 'pytest'
Сгенерировать отчет 'allure generate allure-files -o allure-report'
Открыть отчет 'allure open allure-report'

Стек:
pytest
selenium
requests
allure
config

Структура:
./test/api - api-тесты
./test/ui - ui-тесты
./pages - описание страниц
test_config.ini - настройки для тестов

Полезные ссылки:
Финальный проект по ручному тестированию "Читай-город" https://qa-test2025.yonote.ru/share/0e9f376f-d738-480e-b56a-d3fb3db44be5
